# Fullstack Demo

全栈 Web 应用练习项目，前端 React + 后端 Node.js。

---

## 结构

```
fullstack-demo/
├── frontend/     # React (Vite)
│   ├── src/
│   ├── index.html
│   └── vite.config.js
└── backend/      # Node.js (Express)
    ├── index.js
    └── package.json
```

## 技术栈

- **前端**：React + Vite + ESLint
- **后端**：Node.js + Express

## 运行

```bash
# 后端
cd backend && npm install && node index.js

# 前端
cd frontend && npm install && npm run dev
```
